---
name: Canonical Operative/Agent Phrasing Standard
description: Platform-wide rule for how the 10 Operatives / 24 Agents model is described in UI, PDFs, edge functions and memory
type: preference
---

**Canonical phrasing:** "24 AI agents organized into 10 Operatives".

- **Cards / marketing UI:** show operative counts only.
  Core = 5 AI Operatives, Boost = 7, Pro = 10, Elite = 10 (all 10).
- **PDFs / exports:** may add the agent count for clarity.
  Use "All 10 AI Operatives (24 agents)" for Elite/full-suite copy.
- **Forbidden:** "19 AI agents" (legacy), "8 agents / 12 agents / 16 agents"
  as standalone tier descriptors. Don't claim per-tier raw agent counts on
  marketing cards — operative counts only.
- Capitalization: "AI Operatives" (title case).

**Why:** prevents the long-standing 19-vs-24 drift across PDFs and ensures
the homepage, signup, help, and exports all speak the same numbers.
